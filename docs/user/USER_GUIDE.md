## 1. ภาพรวมของระบบ

GAMS เป็นระบบเว็บแอปพลิเคชันสำหรับจัดการการประเมินผลแบบกลุ่มในรายวิชาระดับมหาวิทยาลัย รองรับผู้ใช้ 3 บทบาท:

### บทบาทในระบบ

| บทบาท                   | หน้าที่หลัก                                            |
| ----------------------- | ------------------------------------------------------ |
| **ผู้ดูแลระบบ (Admin)** | จัดการข้อมูลนักศึกษาและอาจารย์ เลื่อนชั้นปี            |
| **อาจารย์ (Teacher)**   | สร้างกิจกรรม กำหนดเกณฑ์การให้คะแนน ให้เกรดแต่ละกลุ่ม   |
| **นักศึกษา (Student)**  | สร้างกลุ่ม ดูคะแนน ดูเกรด และอ่านความคิดเห็นจากอาจารย์ |

### WorkFlow ภาพรวม

```
┌─────────────────────────────────────────────────────────────┐
│                    ผู้ดูแลระบบ (Admin)                         │
│  1. เพิ่มข้อมูลอาจารย์เข้าระบบ                                   │
│  2. เพิ่มข้อมูลนักศึกษาเข้าระบบ                                  │
│  3. จัดการผู้ใช้ (ค้นหา, ลบ, เลื่อนชั้นปี)                         │
└──────────────────────┬──────────────────────────────────────┘
                       │
          ┌────────────┴────────────┐
          ▼                         ▼
┌─────────────────────┐   ┌─────────────────────┐
│   อาจารย์ (Teacher)   │   │  นักศึกษา (Student)   │
│                     │   │                     │
│ 1. สร้างกิจกรรม       │   │ 1. ดูกิจกรรมที่เปิดรับ    │
│ 2. กำหนดเกณฑ์การให้   │   │ 2. สร้างกลุ่ม           │
│    คะแนน             │   │    + เพิ่มสมาชิก        │
│ 3. เพิ่มอาจารย์ร่วมให้  │   │ 3. รอรับการประเมิน      │
│    คะแนน             │   │ 4. ดูผลคะแนน/เกรด     │
│ 4. ให้เกรดแต่ละกลุ่ม    │   │    บน Dashboard       │
│ 5. ส่งคะแนน (Submit)  │   │                     │
└─────────────────────┘   └─────────────────────┘
          │                         ▲
          │    คะแนนจากอาจารย์ทุกคน     │
          │    ถูกเฉลี่ยรวมเป็น GPA      │
          └─────────────────────────┘
```

## 2. การเข้าสู่ระบบ

### วิธีเข้าสู่ระบบ

1. เปิดเว็บแอปพลิเคชันที่ http://localhost:4300
2. กรอก Username และ Password
3. กดปุ่ม **เข้าสู่ระบบ**
4. ระบบจะนำไปหน้าหลักตาม Role ของผู้ใช้
   - หากข้อมูลไม่ถูกต้อง จะมี **Modal แจ้งเตือน** ขึ้นมา

### ข้อมูลการเข้าสู่ระบบ

| บทบาท       | Username                        | Password       |
| ----------- | ------------------------------- | -------------- |
| ผู้ดูแลระบบ | `Admin`                         | `p@ssw0rd`     |
| อาจารย์     | อีเมล (เช่น teacher1@email.com) | เลขบัตรประชาชน |
| นักศึกษา    | รหัสนักศึกษา (เช่น 64012345)    | เลขบัตรประชาชน |

### หลังเข้าสู่ระบบ

- ระบบจะแสดง Header ด้านบนพร้อมข้อมูลผู้ใช้
  - นักศึกษา: แสดงชื่อเล่น
  - อาจารย์: แสดง "อาจารย์" + ชื่อเล่น
  - ผู้ดูแลระบบ: แสดง "ผู้ดูแลระบบ"
- กดปุ่ม **ออกจากระบบ** ที่ Header เพื่อ Logout

## 3. ผู้ดูแลระบบ (Admin)

### 3.1 หน้าหลัก

หลังเข้าสู่ระบบ จะเห็นหน้า Dashboard พร้อมเมนูจัดการผู้ใช้

### 3.2 จัดการนักศึกษา

เข้าเมนู **จัดการผู้ใช้** > ส่วน **นักศึกษา**

#### สิ่งที่ทำได้:

| การดำเนินการ            | วิธีใช้งาน                                     |
| ----------------------- | ---------------------------------------------- |
| **ดูรายชื่อนักศึกษา**   | แสดงแยกตามชั้นปี (ปี 1-4) พร้อม pagination     |
| **ดูนักศึกษาที่จบแล้ว** | เลื่อนดูส่วน "จบการศึกษา" ด้านล่าง             |
| **ค้นหานักศึกษา**       | พิมพ์รหัสนักศึกษาหรือชื่อในช่องค้นหา           |
| **เพิ่มนักศึกษาใหม่**   | กดปุ่ม "เพิ่มนักศึกษา" > กรอกข้อมูล > บันทึก   |
| **แก้ไขข้อมูลนักศึกษา** | กดไอคอนดินสอ > แก้ไขข้อมูล > บันทึก            |
| **ลบนักศึกษา**          | กดไอคอนถังขยะ > ยืนยันการลบ                    |
| **ลบนักศึกษาทั้งหมด**   | กดปุ่ม "ลบทั้งหมด" > ยืนยัน                    |
| **ลบนักศึกษาตามชั้นปี** | กดปุ่ม "ลบชั้นปีนี้" ที่แต่ละปี > ยืนยัน       |
| **เลื่อนชั้นปีทั้งหมด** | กดปุ่ม "เลื่อนชั้นปี" > ยืนยัน (ปี 1→2→3→4→จบ) |
| **เลื่อนชั้นปีเฉพาะปี** | กดปุ่ม "เลื่อนชั้น" ที่แต่ละปี > ยืนยัน        |

#### ข้อมูลที่ต้องกรอกเมื่อเพิ่ม/แก้ไขนักศึกษา:

- รหัสนักศึกษา (เฉพาะตอนเพิ่ม)
- ชื่อ-นามสกุล
- ชื่อเล่น
- เลขบัตรประชาชน (ใช้เป็นรหัสผ่าน)
- ชั้นปี
- ปีที่เข้าศึกษา

### 3.3 จัดการอาจารย์

เข้าเมนู **จัดการผู้ใช้** > ส่วน **อาจารย์**

#### สิ่งที่ทำได้:

| การดำเนินการ           | วิธีใช้งาน                                  |
| ---------------------- | ------------------------------------------- |
| **ดูรายชื่ออาจารย์**   | แสดงรายชื่อทั้งหมด                          |
| **ค้นหาอาจารย์**       | พิมพ์อีเมลหรือชื่อในช่องค้นหา               |
| **เพิ่มอาจารย์ใหม่**   | กดปุ่ม "เพิ่มอาจารย์" > กรอกข้อมูล > บันทึก |
| **แก้ไขข้อมูลอาจารย์** | กดไอคอนดินสอ > แก้ไขข้อมูล > บันทึก         |
| **ลบอาจารย์**          | กดไอคอนถังขยะ > ยืนยันการลบ                 |

#### ข้อมูลที่ต้องกรอกเมื่อเพิ่ม/แก้ไขอาจารย์:

- ชื่อ-นามสกุล
- ชื่อเล่น
- เลขบัตรประชาชน (ใช้เป็นรหัสผ่าน)
- อีเมล (ใช้เป็น Username สำหรับเข้าสู่ระบบ)

### 3.4 ภาพรวมกิจกรรม (Activity Overview)

เข้าเมนู **จัดการผู้ใช้** > ส่วน **ภาพรวมกิจกรรม**

Admin สามารถดูกิจกรรมทั้งหมดในระบบแบบ read-only:

- ค้นหากิจกรรมตามชื่อหรือกรองตามอาจารย์ผู้สร้าง
- ดูรายละเอียดกิจกรรม: เกณฑ์การให้คะแนน, ผู้ร่วมให้คะแนน, กลุ่มที่เข้าร่วม
- แสดงแบบ paginated (10 รายการต่อหน้า)

> **หมายเหตุ:** Admin ดูกิจกรรมได้อย่างเดียว ไม่สามารถสร้าง แก้ไข หรือลบกิจกรรมได้

## 4. อาจารย์ (Teacher)

### 4.1 หน้าหลัก

หลังเข้าสู่ระบบ จะเห็นหน้า Dashboard พร้อมเมนู:

- **กิจกรรม** - ดูรายการกิจกรรมทั้งหมด
- **สร้างกิจกรรม** - สร้างกิจกรรมใหม่
- **รายชื่อนักศึกษา** - ดูรายชื่อนักศึกษาตามชั้นปี (การ์ดสี Teal)

### 4.2 สร้างกิจกรรม

1. กดปุ่ม **สร้างกิจกรรมใหม่**
2. กรอกข้อมูลกิจกรรม:
   - ชื่อกิจกรรม
   - รายละเอียด
   - วันเริ่มต้น / วันสิ้นสุด
   - ชั้นปีเป้าหมาย (เลือกว่านักศึกษาปีไหนจะเห็นกิจกรรมนี้)
3. **กำหนดเกณฑ์การให้คะแนน** (เพิ่มได้หลายเกณฑ์):
   - ชื่อเกณฑ์ (เช่น ความคิดสร้างสรรค์, การทำงานเป็นทีม)
   - น้ำหนัก (Weight) สำหรับคำนวณ GPA เฉลี่ย
4. **เลือกอาจารย์ร่วมให้คะแนน** (ถ้ามี):
   - เลือกอาจารย์ท่านอื่นที่ต้องการให้ช่วยให้คะแนน
   - อาจารย์ที่ถูกเลือกจะเห็นกิจกรรมนี้ในรายการของตนเอง
5. กดปุ่ม **สร้าง**

### 4.3 ดูรายการกิจกรรม (My Activities)

- แสดงกิจกรรมที่ตนเองสร้าง **และ** กิจกรรมที่ถูกมอบหมายเป็นผู้ร่วมให้คะแนน
- **เรียงตามวันที่สร้างล่าสุด** (กิจกรรมล่าสุดอยู่บนสุด)
- **ค้นหา**: พิมพ์ชื่อกิจกรรมในช่องค้นหาเพื่อกรองผลลัพธ์
- **Pagination**: แสดงหน้าละ 10 รายการ พร้อมปุ่มเปลี่ยนหน้า
- แต่ละกิจกรรมแสดง: **ชื่อ, ชั้นปีเป้าหมาย (badge สีน้ำเงิน), วันที่เริ่ม/สิ้นสุด, จำนวนกลุ่ม**
  - ถ้าไม่ได้กำหนดชั้นปี จะแสดง "ทุกชั้นปี"
- ปุ่มดูความคืบหน้า (ไอคอนดวงตา) - เข้าหน้าดู Progress
- ปุ่มให้คะแนน (ไอคอนดินสอ) - เข้าหน้าให้คะแนน
- ปุ่มลบ (ไอคอนถังขยะ) - แสดงเฉพาะอาจารย์ที่เป็นคนสร้างกิจกรรมเท่านั้น

### 4.4 ดูความคืบหน้ากิจกรรม (Activity Progress)

กดปุ่มดวงตา (ไอคอนสีม่วง) ที่กิจกรรมที่ต้องการดูใน My Activities

#### หน้า Progress แสดงข้อมูล:

| ส่วน                 | แสดงข้อมูล                                                     |
| -------------------- | -------------------------------------------------------------- |
| **ข้อมูลกิจกรรม**    | ชื่อ, รายละเอียด, วันเริ่มต้น, วันสิ้นสุด                      |
| **สถิติรวม**         | จำนวนกลุ่มที่เข้าร่วม, ผู้ให้คะแนนที่ส่งแล้ว, กำลังดำเนินการ   |
| **ตารางผู้ให้คะแนน** | ชื่ออาจารย์, สถานะ (ส่งแล้ว/ร่าง), progress bar, วันที่ส่ง     |
| **คะแนนแต่ละกลุ่ม**  | ชื่อกลุ่ม, เกรดเฉลี่ย (GPA), กดขยายเพื่อดูคะแนนจากแต่ละอาจารย์ |

#### การดูคะแนนแต่ละกลุ่ม:

1. กดที่การ์ดกลุ่มเพื่อขยาย
2. จะเห็นรายชื่อสมาชิก
3. ตารางแสดงเกรดจากแต่ละอาจารย์ แยกตามเกณฑ์
4. ความคิดเห็นจากแต่ละอาจารย์
5. เกรดเฉลี่ยรวม (คำนวณจาก GPA ของทุกอาจารย์)

### 4.5 ให้คะแนน (Grading)

1. กดปุ่มดินสอที่กิจกรรมที่ต้องการใน My Activities
2. **Badge สถานะ** ด้านบนขวาของหน้า:
   - **"รอให้คะแนน"** (พื้นขาว) — ยังให้คะแนนไม่ครบทุกกลุ่ม
   - **"ให้คะแนนแล้ว"** (พื้นเขียว) — ให้คะแนนครบทุกกลุ่มทุกเกณฑ์แล้ว
3. เลือกเกรด (A, B+, B, C+, C, D+, D, F) สำหรับแต่ละกลุ่มและแต่ละเกณฑ์
4. เขียนความคิดเห็น (ถ้ามี) แล้วกดปุ่ม **บันทึก** เพื่อบันทึกเป็น Draft
5. เมื่อพร้อมแล้ว กดปุ่ม **ส่งเพื่ออนุมัติ** ใน Breadcrumb ด้านบน
   - ระบบจะ**บันทึกคะแนนทุกกลุ่มอัตโนมัติ** แล้วส่งเพื่ออนุมัติในขั้นตอนเดียว
   - กลุ่มที่ส่งแล้วจะถูกล็อค ไม่สามารถแก้ไขได้
   - **กลุ่มที่เพิ่มมาหลังการส่ง** ยังสามารถให้คะแนนและกด "ส่งเพื่ออนุมัติ" ได้อีกครั้ง
   - หลังส่งสำเร็จ หน้าจะรีโหลดข้อมูลใหม่อัตโนมัติ

### 4.6 ดูรายชื่อนักศึกษา (Student List)

อาจารย์สามารถดูรายชื่อนักศึกษาได้จากหน้า Teacher Dashboard โดยกดที่การ์ดสี Teal "รายชื่อนักศึกษา"

#### สิ่งที่ทำได้:

| การดำเนินการ           | วิธีใช้งาน                                                                                  |
| ---------------------- | ------------------------------------------------------------------------------------------- |
| **ดูรายชื่อตามชั้นปี** | เลือกแท็บชั้นปี (ปี 1-4) เพื่อดูรายชื่อนักศึกษาแต่ละชั้นปี                                  |
| **ค้นหานักศึกษา**      | พิมพ์รหัสนักศึกษา (studentId), ชื่อ-นามสกุล (fullName), หรือชื่อเล่น (nickname) ในช่องค้นหา |

> **หมายเหตุ:** หน้านี้เป็นแบบ Read-only เท่านั้น อาจารย์ไม่สามารถเพิ่ม แก้ไข หรือลบข้อมูลนักศึกษาได้ การจัดการข้อมูลนักศึกษาต้องดำเนินการโดยผู้ดูแลระบบ (Admin)

### 4.7 การให้คะแนนร่วมกันหลายอาจารย์

- เมื่อกิจกรรมมีอาจารย์ร่วมให้คะแนนหลายคน แต่ละอาจารย์จะให้คะแนนอิสระของตนเอง
- คะแนนจากอาจารย์ทุกคนที่ส่งแล้ว (Submitted) จะถูกเฉลี่ยรวมกัน
- นักศึกษาจะเห็นค่า GPA เฉลี่ยและเกรดสุดท้ายที่คำนวณจากทุกอาจารย์

**ตัวอย่าง:**

- อาจารย์ A ให้เกรด: A (4.00 คะแนน)
- อาจารย์ B ให้เกรด: B+ (3.50 คะแนน)
- GPA เฉลี่ย = (4.00 + 3.50) / 2 = 3.75 → เกรดรวม: **A**

## 5. นักศึกษา (Student)

### 5.1 หน้าหลัก

หลังเข้าสู่ระบบ จะเห็นหน้า Dashboard พร้อมเมนู:

- **กิจกรรม** - ดูกิจกรรมที่เปิดรับ
- **กลุ่มของฉัน** - จัดการกลุ่ม
- **แดชบอร์ด** - ดูคะแนนและเกรด

### 5.2 ดูกิจกรรมที่เปิดรับ

- แสดงกิจกรรมที่ตรงกับชั้นปีของนักศึกษา **เรียงตามวันที่สร้างล่าสุด**
- **กิจกรรมที่ยังไม่มีกลุ่มจะขึ้นก่อนเสมอ** เพื่อให้สร้างกลุ่มได้สะดวก
- ถ้ายังไม่มีกลุ่มในกิจกรรมนั้น จะแสดงปุ่ม **สร้างกลุ่ม**; ถ้ามีกลุ่มแล้วปุ่มจะหายไป

### 5.3 จัดการกลุ่ม (My Groups)

#### สร้างกลุ่มใหม่:

1. กดปุ่ม **สร้างกลุ่มใหม่**
2. เลือกกิจกรรม
3. ตั้งชื่อกลุ่ม
4. เขียนรายละเอียดโปรเจค
5. เลือกสมาชิกในกลุ่ม
6. กดปุ่ม **สร้าง**

#### แก้ไขกลุ่ม (เฉพาะคนที่สร้างกลุ่มเท่านั้น):

- กดไอคอนดินสอที่มุมขวาบนของการ์ดกลุ่ม
- แก้ไขชื่อกลุ่ม, รายละเอียด, หรือเปลี่ยนสมาชิก
- กดบันทึก

#### ลบกลุ่ม (เฉพาะคนที่สร้างกลุ่ม):

- กดไอคอนถังขยะที่มุมขวาบน
- ยืนยันการลบ

#### ดูผลคะแนน:

- แต่ละกลุ่มจะแสดง Badge คะแนน:
  - **GPA x.xx** พร้อม **เกรด** (เช่น GPA 3.50 B+)
  - Badge เกรดจะเปลี่ยนสีตามเกรดที่ได้
  - ถ้ายังไม่มีคะแนน จะแสดง "รอให้คะแนน"

### 5.4 แดชบอร์ดคะแนน (My Score Dashboard)

หน้านี้แสดงข้อมูลคะแนนทั้งหมดของนักศึกษา

#### ส่วนบน - สถิติรวม:

| การ์ด                 | แสดงข้อมูล                                |
| --------------------- | ----------------------------------------- |
| **Total Activities**  | จำนวนกิจกรรมทั้งหมดที่เข้าร่วม            |
| **Activities Graded** | จำนวนกิจกรรมที่ได้รับคะแนนแล้ว            |
| **GPA เฉลี่ย**        | ค่า GPA เฉลี่ยจากทุกกิจกรรม (0.00 - 4.00) |

#### ส่วนกลาง - ตารางสรุปเกรด:

| คอลัมน์ | แสดงข้อมูล                                  |
| ------- | ------------------------------------------- |
| กิจกรรม | ชื่อกิจกรรม                                 |
| กลุ่ม   | ชื่อกลุ่มที่เข้าร่วม                        |
| GPA     | ค่า GPA ของกิจกรรมนั้น                      |
| เกรด    | เกรดตัวอักษร (A, B+, B, ...) พร้อมสีตามเกรด |
| สถานะ   | Graded (ให้คะแนนแล้ว) หรือ Pending (รอ)     |

#### ส่วนล่าง - รายละเอียดแต่ละกิจกรรม:

- **เกรดรวม** - แสดงเกรดตัวใหญ่พร้อมค่า GPA สีตามเกรด
- **คะแนนรายเกณฑ์** - แต่ละเกณฑ์แสดง GPA และเกรดย่อย
- **ความคิดเห็นจากอาจารย์** - แสดงชื่ออาจารย์และความคิดเห็น

## 6. ระบบเกรดและ GPA

### 6.1 เกรดตามมาตรฐานการศึกษาไทย

ระบบใช้เกรดตามระบบมหาวิทยาลัยไทย:

| เกรด   | ค่า GPA | ความหมาย   | สีที่แสดง           |
| ------ | ------- | ---------- | ------------------- |
| **A**  | 4.00    | ดีเยี่ยม   | เขียว (Emerald)     |
| **B+** | 3.50    | ดีมาก      | เขียวอมฟ้า (Teal)   |
| **B**  | 3.00    | ดี         | น้ำเงิน (Blue)      |
| **C+** | 2.50    | ค่อนข้างดี | เหลืองอำพัน (Amber) |
| **C**  | 2.00    | ปานกลาง    | ส้ม (Orange)        |
| **D+** | 1.50    | อ่อน       | แดงอ่อน             |
| **D**  | 1.00    | อ่อนมาก    | แดง                 |
| **F**  | 0.00    | ตก         | แดงเข้ม             |

### 6.2 วิธีคำนวณ GPA เฉลี่ย

การคำนวณ GPA เฉลี่ยของแต่ละกลุ่มทำดังนี้:

**ขั้นตอนที่ 1:** แปลงเกรดตัวอักษรเป็นค่า GPA

- เช่น อาจารย์ A ให้เกรด A → 4.00, อาจารย์ B ให้เกรด B+ → 3.50

**ขั้นตอนที่ 2:** เฉลี่ยค่า GPA ต่อเกณฑ์ (ถ้ามีหลายอาจารย์)

- เช่น เกณฑ์ "ความคิดสร้างสรรค์": (4.00 + 3.50) / 2 = 3.75

**ขั้นตอนที่ 3:** คำนวณ GPA ถ่วงน้ำหนัก

- GPA รวม = ผลรวม(GPA เกณฑ์ x น้ำหนัก) / ผลรวม(น้ำหนัก)

**ขั้นตอนที่ 4:** แปลง GPA กลับเป็นเกรดตัวอักษร

| ค่า GPA เฉลี่ย | เกรดที่ได้ |
| -------------- | ---------- |
| >= 3.75        | A          |
| >= 3.25        | B+         |
| >= 2.75        | B          |
| >= 2.25        | C+         |
| >= 1.75        | C          |
| >= 1.25        | D+         |
| >= 0.50        | D          |
| < 0.50         | F          |

### 6.3 ตัวอย่างการคำนวณ

สมมติกิจกรรมมี 2 เกณฑ์ และมีอาจารย์ 2 คนร่วมให้คะแนน:

**เกณฑ์ 1: ความคิดสร้างสรรค์ (น้ำหนัก 1.0)**

- อาจารย์ A ให้: A (4.00)
- อาจารย์ B ให้: B+ (3.50)
- GPA เฉลี่ยเกณฑ์ 1 = (4.00 + 3.50) / 2 = **3.75**

**เกณฑ์ 2: การทำงานเป็นทีม (น้ำหนัก 1.0)**

- อาจารย์ A ให้: B (3.00)
- อาจารย์ B ให้: B+ (3.50)
- GPA เฉลี่ยเกณฑ์ 2 = (3.00 + 3.50) / 2 = **3.25**

**GPA รวม** = (3.75 x 1.0 + 3.25 x 1.0) / (1.0 + 1.0) = **3.50**
**เกรดรวม** = B+ (เพราะ 3.50 >= 3.25)

## 7. คำถามที่พบบ่อย

### ทั่วไป

**Q: เข้าสู่ระบบไม่ได้ ขึ้น "User not found in system"**

> ตรวจสอบว่า Admin ได้เพิ่มข้อมูลผู้ใช้เข้าระบบแล้ว และใช้ Username/Password ถูกต้อง

**Q: ลืมรหัสผ่าน**

> รหัสผ่านคือเลขบัตรประชาชน ติดต่อผู้ดูแลระบบเพื่อตรวจสอบ

### อาจารย์

**Q: ไม่เห็นกิจกรรมที่ถูกมอบหมายให้ร่วมให้คะแนน**

> กิจกรรมที่อาจารย์ท่านอื่นเพิ่มคุณเป็นผู้ร่วมให้คะแนน จะแสดงในรายการ My Activities หากไม่เห็น ให้ลอง Refresh หน้าเว็บ

**Q: ส่งคะแนนแล้วแก้ไขไม่ได้**

> หลังกดปุ่ม "ส่งเพื่ออนุมัติ" คะแนนจะถูกล็อค ไม่สามารถแก้ไขได้ ให้ตรวจสอบให้ดีก่อนส่ง

**Q: ลบกิจกรรมไม่ได้**

> เฉพาะอาจารย์ที่เป็นคนสร้างกิจกรรมเท่านั้นที่สามารถลบได้ ผู้ร่วมให้คะแนนจะไม่เห็นปุ่มลบ

### นักศึกษา

**Q: แดชบอร์ดแสดง "-" ไม่มีคะแนน**

> คะแนนจะแสดงก็ต่อเมื่ออาจารย์กด "ส่งเพื่ออนุมัติ" แล้วเท่านั้น ถ้าอาจารย์ยังบันทึกเป็น Draft อยู่ จะยังไม่แสดง

**Q: สร้างกลุ่มไม่ได้ ขึ้น "Student already in a group"**

> นักศึกษาแต่ละคนสามารถอยู่ในกลุ่มได้เพียง 1 กลุ่มต่อ 1 กิจกรรม หากต้องการเปลี่ยนกลุ่ม ให้ลบกลุ่มเดิมก่อน (เฉพาะคนที่สร้างกลุ่ม)

**Q: แก้ไขกลุ่มไม่ได้**

> เฉพาะนักศึกษาที่เป็นคนสร้างกลุ่มเท่านั้นที่สามารถแก้ไขหรือลบกลุ่มได้ จะเห็นไอคอนดินสอและถังขยะที่มุมขวาบนของการ์ด

**Q: สีของเกรดหมายความว่าอะไร**

> สีช่วยให้เห็นภาพรวมผลการเรียนได้ง่าย:
>
> - สีเขียว = ดี (A, B+, B)
> - สีเหลือง/ส้ม = ปานกลาง (C+, C)
> - สีแดง = ต้องปรับปรุง (D+, D, F)
